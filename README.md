# Kishna_mitra
🌾 Smart Farming AI Platform

An AI-powered smart farming assistance platform designed to help farmers make better agricultural decisions using data-driven insights, weather intelligence, and AI-based crop advisory tools.

This platform provides farmers with tools to improve crop productivity, detect pests early, optimize fertilizer usage, and receive real-time weather alerts.

🚀 Features

🌱 Crop Recommendation System
Suggests suitable crops based on location, soil type, and seasonal conditions.

💧 Smart Irrigation Guidance
Provides irrigation schedules based on weather forecasts and crop growth stages.

🐛 Pest & Disease Detection
Farmers can upload crop images to detect diseases and receive treatment recommendations.

🌾 Fertilizer Optimization
Recommends appropriate fertilizers and dosage based on crop requirements.

🌦 Weather Alerts
Provides weather forecasts and alerts to help farmers plan farming activities.

💬 AI Chatbot Assistance
Interactive chatbot to help farmers with farming-related questions.

📱 OTP Authentication
Secure mobile login using OTP authentication.

📶 Offline Support (PWA)
Works even with low internet connectivity using Progressive Web App technology.

❤️ Farmer Relief Donation Initiative
Allows users to support farmers facing crop losses through a donation system.
